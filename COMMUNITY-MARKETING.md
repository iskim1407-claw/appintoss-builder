### 📝 콘텐츠 초안 — "사이드프로젝트 공유" (복붙 가능)

---

**제목:** 토스 미니앱 노코드 빌더 만든 후기 (1인 개발, 8주)

---

안녕하세요, 평일엔 회사 다니고 퇴근 후/주말에 사이드 프로젝트 하는 개발자입니다.

**🎯 만든 것**
토스 미니앱을 노코드로 만들 수 있는 빌더입니다.
드래그앤드롭으로 컴포넌트 배치하고, 토스 심사용 ZIP 파일까지 자동 생성해줍니다.

**💡 왜 만들었나**
- 토스 미니앱 만들려면 TDS(토스 디자인 시스템) 배워야 함
- React 개발자 아니면 진입장벽 높음
- "비개발자도 토스에 앱 올릴 수 있으면 좋겠다" 생각

**🛠️ 기술 스택**
- Next.js 14 (App Router)
- Craft.js (비주얼 에디터 핵심)
- TailwindCSS
- Turso (SQLite, 프로젝트 저장)
- Vercel 배포

**📊 현재 상태**
- 33종 컴포넌트 완성
- 심사 가이드라인 33개 항목 자동 검증
- 실제로 제가 만든 앱 2개 토스 심사 제출함 (월급루팡 계산기, 소비성향 테스트)
- 아직 심사 결과 대기 중

**🤔 가장 어려웠던 점**
1. Craft.js 문서가 친절하지 않음... 소스코드 까보면서 이해함
2. 토스 TDS 컴포넌트를 Craft.js 호환되게 래핑하는 게 생각보다 복잡
3. "심사 통과할 수 있는 앱"의 기준을 정의하는 것

**📈 배운 점**
- 노코드 빌더 만드는 건 결국 "제약을 어디까지 둘 것인가"의 문제
- 처음엔 자유도 높게 했다가, 오히려 유저가 뭘 해야 할지 모름
- 템플릿 → 수정 → 제출 흐름이 훨씬 전환율 좋음

### 📝 콘텐츠 초안 A — "사이드프로젝트 후기" (복붙 가능)

---

**제목:** 퇴근 후 2시간씩, 토스 미니앱 만들어서 출시 준비 중인 썰

---

IT 대기업 다니는 5년차입니다.
본업은 프론트엔드 개발인데, 회사 일만 하니까 뭔가 허전해서 사이드 시작했어요.

**뭘 만들었나**
토스 앱 안에서 돌아가는 미니앱 만드는 노코드 빌더입니다.
(토스 열면 "앱인토스"라고 있는데, 거기 들어가는 앱들)

**왜 토스 미니앱?**
- 토스 유저가 3000만명임
- 앱스토어 심사보다 훨씬 빠름 (3-5일)
- 설치 없이 바로 사용 → 진입장벽 낮음
- 돌돌디라는 앱이 월매출 2.1억 찍었다는 기사 보고 관심 생김

**과정**
- 평일: 퇴근 후 2시간 (피곤하면 1시간만)
- 주말: 반나절
- 총 8주 정도 걸림

**현재 상태**
- 제가 만든 미니앱 2개 심사 제출함
- "월급루팡 계산기" (월급 대비 실제 일한 시간 계산)
- 아직 심사 결과 대기 중

**느낀 점**
- 회사 코드랑 다르게 내 맘대로 할 수 있어서 재밌음
- 대신 혼자 다 해야 해서 번아웃 조심해야 함
- 출퇴근 시간에 기획하고, 점심시간에 디자인 고민하면 의외로 시간 남

**앞으로 계획**
- 심사 통과하면 그거 가지고 바이럴 노려볼 예정
- 다른 사람들도 쉽게 만들 수 있게 빌더 공개함
- 혹시 비개발자인데 토스 미니앱 만들어보고 싶은 분 계시면 써보시라고...

질문 있으시면 댓글 주세요.

취준 중인데 포폴 뭐 넣을지 고민하다가 발견한 거 공유함

**토스 앱인토스라고 있음**
- 토스 앱 안에서 돌아가는 미니앱 서비스
- 개인도 앱 만들어서 올릴 수 있음
- 3000만명이 쓰는 토스에 내 앱 노출됨

**코딩 못해도 되는 이유**
- 노코드 빌더 있음 (드래그앤드롭)
- 토스 디자인이 기본 적용돼서 UI 고민 X
- 만들고 → 심사 제출 → 통과하면 앱 올라감

**뭘 만들 수 있냐면**
- MBTI 테스트 같은 퀴즈앱
- 밸런스 게임
- 가계부/계산기
- 소상공인이면 가게 소개/예약 앱

**포폴에 어떻게 써먹냐**
- "토스 미니앱 기획 및 출시"
- 실사용자 있는 실서비스 경험 → 면접에서 어필 가능
- 기획/PM 지망이면 더 좋음

**어디서 하냐**
→ 댓글로 알려줄게

---

진짜 한번 해보면 2-3시간이면 만들 수 있음
면접에서 "토스에 앱 출시해봤다"고 하면 관심 가질 듯

**빌더 이름은**

안녕하세요, 앱인토스에 앱 출시 준비 중인 회원입니다.

---

## 1️⃣ 앱인토스란?

토스 앱 안에서 돌아가는 미니앱 서비스입니다.
- 토스 유저 3000만명에게 노출 가능
- 앱스토어 없이 바로 사용자 접근
- 토스페이 결제 연동 가능

**성공 사례**
- 돌돌디: 월매출 2.1억
- 이사모아: 이사 견적 서비스

---

## 2️⃣ 준비물

- 토스 개발자 계정 (무료)
- 앱 아이디어
- 개발 능력 또는 노코드 빌더

---

## 3️⃣ 앱 만드는 방법

### 방법 A: 직접 개발
- React + TDS (토스 디자인 시스템) 필요
- 개발자 대상
- 자유도 높음

### 방법 B: 노코드 빌더 사용
- 드래그앤드롭으로 제작
- 비개발자도 가능
- 제가 사용해본 빌더 중 "앱인토스 빌더"가 가장 쉬웠어요
- (검색하시면 나옵니다)

---

## 4️⃣ 심사 과정

1. 앱 제작 완료
2. ZIP 파일로 패키징
3. 토스 개발자 콘솔에 업로드
4. 심사 대기 (평균 3-5일)
5. 승인되면 토스 앱에 노출

**심사 탈락 주요 사유**
- 저작권 위반 (폰트, 이미지)
- 개인정보 수집 동의 누락
- 광고 과다
- TDS 가이드라인 미준수

---

## 5️⃣ 수익화 방법

- 토스페이 결제 연동 (수수료 3.5%)
- 광고 삽입 (토스 정책 확인 필요)
- 유료 콘텐츠/기능

---

## 6️⃣ 꿀팁

1. 처음엔 간단한 앱부터 (퀴즈, 계산기 등)
2. TDS 컴포넌트 그대로 쓰면 심사 통과율 높음
3. 빌더 쓰면 심사 가이드라인 자동 체크됨
4. 심사 전 "미리보기"로 꼭 확인

### vs 앱스토어 출시

- **소상공인**: 가게 앱 간편하게
- **마케터**: 이벤트/캠페인 앱
- **학생**: 포트폴리오용 실서비스

## 3. 앱인토스 시작하는 방법

### Step 1: 토스 개발자 계정 만들기

## 2️⃣ 콘텐츠 전략

### 바이럴 콘텐츠 공식

```
[충격적 숫자] + [짧은 시간] + [결과물] = 바이럴
```

예시:
- "0원으로 3천만 유저 앱 만들기"
- "5분만에 토스 퀴즈앱 완성"
- "코드 0줄로 월급루팡 계산기 만들기"

### 콘텐츠 유형별 전략

#### 🎬 영상 콘텐츠 (60초 이내)
1. **타임랩스 빌드** — 빈 캔버스 → 완성된 앱 (60초 압축)
2. **Before/After** — "외주 견적 500만원" vs "내가 직접 0원"
3. **클릭 ASMR** — 드래그앤드롭 소리 + 컴포넌트 배치
4. **리액션 영상** — 토스에서 내 앱 실행되는 순간

#### 📝 텍스트 콘텐츠
1. **빌드로그** — "토스 미니앱 출시까지 7일 일지"
2. **수익 공개** — "토스 미니앱으로 첫 수익 나온 썰" (심사 통과 후)
3. **비교 분석** — "앱 개발 외주 vs 노코드 빌더 vs 직접 코딩"
4. **실패담** — "토스 심사 3번 떨어진 이유" (공감 유발)

#### 🧪 바이럴 콘텐츠 템플릿

**X/Twitter 쓰레드 구조:**
```
1/ 어제 퇴근 후 2시간, 토스 미니앱 하나 만들었다

2/ 개발자 아님. 기획자임. 코드 1줄도 안 씀

3/ 사용한 도구: 앱인토스 빌더 (무료)

4/ 만든 앱: [스크린샷]

5/ 심사 제출까지 걸린 시간: 2시간

6/ 이제 3천만 토스 유저한테 노출될 예정

7/ 만드는 과정 궁금하면 🔄

[빌드 영상 GIF 첨부]
```

**블라인드 게시글 구조:**
```
제목: 퇴근 후 토스 미니앱 런칭한 썰

- 본업: IT 대기업 PM
- 코딩 실력: 없음
- 사용 도구: 앱인토스 빌더
- 걸린 시간: 이틀
- 결과: 토스에서 내 앱 보인다

[스크린샷]

사이드프로젝트 하고 싶은데 개발 못해서 못하는 분들 참고하세요
```

## 3️⃣ 타겟 오디언스 페르소나

### 페르소나 A: 바이브코더 김개발 (25-35세)
- **특징:** 커서/클로드 코드 사용, X 트위터 활발, 사이드프로젝트 욕구
- **페인포인트:** "토스 미니앱 만들어보고 싶은데 TDS 배우기 귀찮다"
- **훅:** "TDS 디자인 시스템 빌트인. 드래그만 하면 토스 네이티브 룩"
- **채널:** X, 디스코드, Hacker News

### 페르소나 B: 기획자/PM 이기획 (28-40세)
- **특징:** 아이디어 많음, 개발 리소스 없음, 외주 비용 고민
- **페인포인트:** "개발자한테 부탁하기 미안함, 외주는 비쌈"
- **훅:** "개발자 없이 프로토타입 → 실제 앱까지"
- **채널:** 블라인드, 브런치, 오픈카톡

### 페르소나 C: 1인 창업자 박스타트 (25-45세)
- **특징:** MVP 빨리 검증하고 싶음, 예산 적음
- **페인포인트:** "3천만 유저한테 테스트해보고 싶다"
- **훅:** "토스페이 연동 가능. 결제 붙이기까지 10분"
- **채널:** 스타트업 커뮤니티, Product Hunt

### 페르소나 D: 대학생/취준생 최열정 (20-27세)
- **특징:** 포트폴리오 필요, 시간 많음, 돈 없음
- **페인포인트:** "취업용 프로젝트 뭐 하지"
- **훅:** "토스 미니앱 런칭 = 실서비스 경험. 이력서에 쓸 수 있음"
- **채널:** 에브리타임, 캠퍼스픽, 링커리어

## 4️⃣ 성장 루프 설계

### 핵심 아이디어
> **빌더로 만든 앱 자체가 빌더 홍보가 되는 구조**

### 루프 1: "Powered by" 뱃지
```
[유저가 앱 제작] 
    → [앱 하단에 "앱인토스 빌더로 제작" 뱃지 자동 삽입]
    → [토스 유저가 앱 사용 중 뱃지 클릭]
    → [빌더 랜딩페이지 이동]
    → [새 유저 유입]
```

**구현:**
- 무료 플랜: "앱인토스 빌더로 제작" 뱃지 필수
- 프로 플랜: 뱃지 제거 옵션

### 루프 2: 템플릿 마켓플레이스
```
[파워유저가 템플릿 제작]
    → [템플릿 공유/판매]
    → [다른 유저가 템플릿 사용]
    → [빌더 노출 증가]
    → [신규 유저 유입]
```

### 루프 3: 쇼케이스 갤러리
```
[유저가 앱 제작]
    → [빌더 갤러리에 자동/수동 등록]
    → ["이 앱 어떻게 만들었어요?" → 빌더 링크]
    → [신규 유저 유입]
```

### 루프 4: 바이럴 퀴즈/테스트 앱
```
[유저가 바이럴 퀴즈앱 제작 (MBTI, 밸런스게임 등)]
    → [토스 유저들 퀴즈 공유]
    → [결과 화면에 "나도 퀴즈 만들기" CTA]
    → [빌더로 유입]
```

**핵심:** 바이럴 퀴즈/테스트 앱이 가장 강력한 성장 루프. 이미 월급루팡 계산기, 소비 성향 테스트 제작 완료 → 이걸 레퍼런스로 활용

| D5 | 블라인드 IT게시판 "사이드프로젝트 공유" 글 |
| D6 | 브런치 글 1개 "개발자 없이 토스 앱 만든 이야기" |

### 📅 1개월차: 콘텐츠 루틴

**주간 루틴 (자동화):**
- **월:** X 쓰레드 1개 (빌드로그 or 팁)
- **수:** 유튜브 Shorts 1개
- **금:** 커뮤니티 참여 (디스코드/오픈카톡 답변)
- **일:** 주간 리뷰, 다음 주 콘텐츠 준비

**월간 목표:**
- X 팔로워: 100명
- 빌더 방문자: 500명
- 심사 제출 앱: 5개 (본인 + 초기 유저)

### 📅 3개월차: 확장

**2개월차:**
- Product Hunt 런칭 준비 (에셋, 설명, 스크린샷)
- 벨로그/티스토리 SEO 글 5개
- 유튜브 채널 개설 (Shorts 모음)

**3개월차:**
- Product Hunt 런칭 실행
- 템플릿 마켓플레이스 베타 오픈
